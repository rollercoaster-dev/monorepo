# Badges

## Badge Data Model
- ID, title, description, criteria, tags, level
- Evidence attachments: images, audio, video, documents
- Metadata: timestamps, issuer, revocation status

## Badge Metadata
- Verifiable Credentials (VC) format
- Decentralized Identifiers (DIDs) integration
- Privacy settings: public, private, selective disclosure

## Evidence Collection
- UI workflows for uploading and annotating evidence
- Offline capture and syncing of media-rich evidence
- Placeholder for evidence verification flows

## Future Extensions
- Badge templating, badge families, and skill trees
