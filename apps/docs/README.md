# Documentation Source

This directory contains the source content for the [Rollercoaster.dev GitHub Wiki](https://github.com/rollercoaster-dev/monorepo/wiki).

## How it works

1.  Edit the markdown files in this directory.
2.  Commit and push to `main`.
3.  A GitHub Action automatically syncs these files to the repository's Wiki.

## Structure

- `Home.md`: The Wiki homepage.
- `_Sidebar.md`: The Wiki navigation sidebar.
- `vision/`, `architecture/`, etc.: Content sections.
