# AI Support

## Scope of AI Assistance
- Badge scaffold generation
- Journaling and motivational prompts
- Learning nudges and routine suggestions
- Plain language transformation and accessibility assistance
- Optional mentor/teacher support tools

## Guardrails & Limitations
- AI suggestions require user validation
- Respect user privacy; process data locally or with consent
- Rate limits and performance considerations for local devices

## Extensibility
- Plugin API for new AI modules
- Configuration for model selection and resource limits
